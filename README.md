# Shreya Mishra Interactive Portfolio

Static interactive portfolio built with HTML, CSS, and JavaScript.

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Files

- `index.html` - portfolio content and sections
- `styles.css` - responsive layout, animated avatar, cards, and interaction states
- `script.js` - canvas animation, filters, counters, mobile nav, and motion toggle
- Resume links point to the public Google Drive resume URL
